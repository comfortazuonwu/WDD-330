Chapter 5:Objects

  • An object is like a dictionary or a database of values.
  • If a property's value is a function, the property is a method.
  • An object can be mutable even if it was declared using const.
  • The "in" operator and hasOwnProperty() can be used to check if an object has a particular property.
  • A nested object is an object that contains other objects. An object can also be passed as a parameter to a function

Chapter 6: DOM

  • DOM is an interface that allows us to use JavaScript to interact with a web page
  • DOM treats everything on a webpage like a node. Every part of the HTML document is a child node of the HTML tag which is the root node.
  • The getAttribute method returns the value of the attribute while the setAttribute changes the value of the element's attribute.
  • The style property can be used to update the CSS of the web page.

Chapter 7: Events

  • Events provide the link between the webpage and the user interactions.
  • When an event triggers an event handler, a callback function is called.
  • The most common types of events are mouse, keyboard and touch events.
  • The preventDefault method of the event object can stop a default behavior happening. Some events do not allow the default behavior to be prevented.
  • The default behaviour of event propagation is bubbling. Bubbling can be stopped from occuring by adding the event.stopPropagation method
