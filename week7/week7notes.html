Chapter 11: Further Functions

  • Meomization is a useful feature in functions. It saves a result of a function in a cache property and it can be resued when the same arguments are used again.
  • An IFFE is an anonymous function that is invoked as soon as it is defined.
  • A recursive function invokes itself until a certain condition is met.
