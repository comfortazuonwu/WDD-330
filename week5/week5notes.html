Chapter 10: Debugging

  • An exception is an error that produces a return value that helps the program deal with the error.
  • An exception produces a stack trace which is a sequence of methods or functions that lead to the cause of problems.
  • Debugging is the process of finding out where bugs occur and fixing them.
  • Breakpoints can be very useful when trying to track down bugs in your code.
  • Testing your code makes it less brittle and helps identify bugs early.
